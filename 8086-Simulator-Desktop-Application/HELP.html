COMPONENTS

  1. SEGMENT INITIALIZER PANEL

    This panel is used to change the values of segment registers(CS, SS, DS, ES).

  2. FLAG REGISTER PANEL

    It display the status of Flag Register.

  3. REGISTER PANEL

    It display the content of all registers.

  4. MEMORY INITIALIZER PANEL


    It is used to feed data in memory location ranging from [00000 - FFFFF].

    Direction To Use: Provide the segment address, offset and data in the corresponding text box and click on SET button.


  5. MEMORY VIEWER PANEL

    It is used to view the data in the memory.

    Direction To Use: Provide the segment address, offset and the count(number of locations to be accessed from the given segment address and offset) in the corresponding text box and click on the VIEW button.


  6. DEBUGGER PANEL

    It provide mechanism to debug the code.

    Direction To Use
    1. Write the code.
    2. Make sure to save the file.
    3. Hit Debug button which will enable the Execute and STOP button.
    4. Hit the execute button to execute/run the line number shown in the Debugger Panel.
    5. Hit the STOP button to stop debugging.

    Note:
    • Debugger will stop automatically after executing HLT instruction.
    • Use STOP button to stop debugging in between.


KEY POINTS

  1. Hexadecimal Values

    All hexadecimal values must end with 'h' or 'H'
    Note: In order to use AH, BH, CH, DH as hexadecimal values put a '0' in the prefix(0AH, 0BH, 0CH,0DH) else they will be considered as registers.


  2. Loop/Jump/Call

    In order to jump/call to a specific line of code create a label.
    Label is any string(alpha-numeric) ending with ':', for example: "LOOP:", "HERE:" etc.

    Example of a program using label

     MOV AX, 2050h
    JMP STOP
    STOP: HLT


  3. How to use comments in code

    Every thing after ';' symbol will be considered as comment in the code.

    Example

     MOV AH, 00H ;Move 00H to AH
    MVI BH, 11H ;Move 11H to BH
    ADD AH, BH ;AH = AH + BH
    HLT ;Stop


  4. Empty lines

    All empty lines in between the codes will be ignored.
